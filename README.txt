Hướng dẫn chạy code 

I.cơ sở dữ liệu. (MongoDB)
- tải và cài đặt mongoDB compass.
- tạo database đặt tên là BanHang.
- tạo các collection. (users, comments, contacts, products, cates)
- mở file database và export file json vào collection tương ứng. (vd:collection users import file BanHang.users.json)
II.chạy source code 
- chạy lệnh "npm install" để cài đặt các gói module.
- chạy lệnh "npm start" sau đó vào browser truy cập "localhost:4000".
